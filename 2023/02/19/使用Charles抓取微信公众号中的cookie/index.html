使用Charles抓取微信公众号中的cookie

Ayden
2023-02-19
tools

配置代理服务器:
在Charles中，选择“Proxy”菜单，然后选择“Proxy Settings”。在弹出的窗口中，选择“Proxy”选项卡，然后选择“Enable transparent HTTP proxying”和“Enable transparent HTTPS proxying”。

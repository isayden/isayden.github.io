在 Markdown 中插入表情可以使用 Emoji 表情符号。这些表情符号是基于 Unicode 编码的，可以在 Markdown 中直接使用。

下面是一些常用的 Emoji 表情符号：

  • :smile: 表示微笑
  • :heart: 表示爱心
  • :+1: 表示赞同
  • :-1: 表示反对
  • :fire: 表示火焰
  • :rocket: 表示火箭
  • :tada: 表示庆祝
